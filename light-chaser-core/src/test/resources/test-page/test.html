    富士山五合目

    富士山是日本国内的最高峰，也是世界上最大的活火山之一，主峰海拔高达3776公尺。富士山山体呈圆锥状，太古时代反复喷发堆积形成山体，火山喷发物层层堆积，成为锥状成层火山山顶有巨大的火山口，直径约800米，深约200米。富士山作为日本民族的象征之一，在全球都享有盛誉。富士山被日本人民誉为“圣岳”，是日本民族引以为傲的象征
    特别提醒：若因天气原因无法登五合目，将该为前往【富士山一合目】游览。

    或  忍野八海

    忍野八海是位于山梨县忍野村的涌泉群。富士山融化的雪水经过80年的过滤，成为了如今这8处涌出的泉水。为国家指定天然记念物、名水百选、新富岳百景之一。包括了出口池、御釜池、底�i池、铫子池、涌池、浊池、镜池、菖蒲池。

    或  河口湖

    富士五湖中最低的湖，湖畔约有501颗枫叶树，秋日红叶之盛构成了以红叶回廊为首的独特景点。红、黄色点缀的湖畔使河口湖更显迷人的魅力，湖水与红叶相映形成美丽的景色，从山林到湖畔，迷人的红枫紧紧相接，让人目不暇接。
